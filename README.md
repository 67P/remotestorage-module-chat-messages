# remoteStorage Module: Chat Messages

Stores chat messages in daily archive documents.

Please feel free to open GitHub issues for questions, feature requests,
protocol proposals, and whatever else you like.

## Protocols

### Currently supported

* IRC
* XMPP

### Planned

* Mattermost
* Matrix
* Slack
* ...
